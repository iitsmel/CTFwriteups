# Base64 Encoding

Base64 encoding is a way to represent binary data as text.

## Use

* Encode:

    * The [base64](http://linux.die.net/man/1/base64) command line program

* Decode:

    * Online [here](http://www.base64decode.org/)

    * The [base64](http://linux.die.net/man/1/base64) command line program
